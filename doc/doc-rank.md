## Tape.rank
> Rank Module

#### Simple [view src](../examples/demo)
```js
// config rank
Tape.rank.setRankKey('score');
// create rank view
Tape.rank.createRankView()
// show rank
Tape.rank.showRank(ui.rankUI.uiView);
// hide rank
Tape.rank.hideRank();
// set self rank score
Tape.rank.setRankScore('score', 1024, '---------');
```
